# Rodin candidate intake

Raw Rodin/Hyper3D generated assets belong here only after preserving:

- prompt/source image hash;
- Rodin model/tier/settings;
- task/download receipt IDs without secrets;
- export files and hashes;
- terms/account-plan snapshot hash;
- license/commercial-use status.

Assets in this directory are candidate-only until moved through `assets_src/production/` by the production acceptance gates.
